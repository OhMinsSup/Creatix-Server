# Creatix

> creatix는 창작자들을 위한 서비스 플랫폼 입니다. 아직 개발 단계입니다.
> 이 프로젝트는 오픈소스로 진행 됩니다.

# On the backend:
 - Koa - Typescript (GraphQL Server ❤️)

- Typeorm - Typescript (ORM 💪🏻)

- JWT (Auth 🔑)

- Postgresql (DB 📃)
