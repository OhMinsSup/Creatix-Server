# Creatix

> creatix는 창작자들을 위한 서비스 플랫폼 입니다. 현재 서비스는 한국어로만 제공할 예정입니다.

# On the backend:
 - Koa - typescript (Server ❤️)

- Typeorm - typescript (ORM 💪🏻)

- JWT (Auth 🔑)

- Postgresql (DB 📃)

- Jest - supertest (Test Framework 🎣)
